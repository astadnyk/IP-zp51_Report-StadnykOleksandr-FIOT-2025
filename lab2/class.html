Селектори класу

Стилізація елементів за класом:

.container {
    width: 80%;
    margin: 0 auto;
}
.button {
    background-color: blue;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
}
.alert {
    color: red;
    font-weight: bold;
}

Елементи з класом "container" матимуть ширину 80% і будуть центровані на сторінці. Елементи з класом "button" матимуть синій фон, білий текст, відступи, відсутність меж і закруглені кути. Елементи з класом "alert" матимуть червоний колір тексту і жирне накреслення.

Ці приклади демонструють, як можна використовувати селектори класів для застосування стилів до конкретних груп елементів на сторінці.
