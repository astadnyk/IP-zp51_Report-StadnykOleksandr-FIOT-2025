Способи підключення стилів

CSS можна підключати трьома способами:

  1. Inline - стилі додаються безпосередньо до HTML елемента за допомогою атрибута: style
    Приклад: 
                <p style="color: blue; font-size: 20px;">Текст</p>
  2. Internal - стилі визначаються всередині HTML документа в секції: <style>
    Приклад: 
                <style>
                p {
                    color: green;
                    font-size: 18px;
                }
            </style>
  3. External - стилі зберігаються в окремому CSS файлі, який підключається до HTML документа за допомогою тега: <link>
    Приклад: 
                <link rel="stylesheet" href="styles.css">
