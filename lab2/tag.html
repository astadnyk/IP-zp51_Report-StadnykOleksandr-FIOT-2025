Селектори тегу

Стилізація елементів за тегом:

header {
    width: 100%;
}
footer {
    width: 100%;
    font-family: Arial, sans-serif;
}

Всі елементи матимуть ширину 100%, а футер буде використовувати шрифт Arial.

Ще приклад для абзаців:

p {
    line-height: 1.5;
    color: black;
}

Всі абзаци матимуть міжрядковий інтервал 1.5 і чорний текст.

Ще приклад для заголовків:

h1 {
    font-size: 14px;
    color: blue;
}

Всі заголовки першого рівня будуть мати розмір шрифту 14px і синій колір.

Ці приклади демонструють, як можна використовувати селектори тегів для застосування стилів до всіх елементів певного типу на сторінці.
