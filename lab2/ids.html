Селектори ідентифікаторів

Стилізація елементів за id:

#main-header {
    background-color: green;
    color: white;
    padding: 10px;
}
#footer {
    background-color: #333;
    color: white;
    text-align: center;
    padding: 5px;
}

Елемент з id "main-header" матиме зелений фон, білий текст і відступи по 10px. Елемент з id "footer" матиме темно-сірий фон, білий текст, центрований текст і відступи по 5px.

Ще приклад для елемента з id "sidebar":

#sidebar {
    width: 250px;
    background-color: yellow;
    padding: 15px;
}

Елемент з id "sidebar" матиме ширину 250px, жовтий фон і відступи по 15px.

Ці приклади демонструють, як можна використовувати селектори ідентифікаторів для застосування стилів до конкретних елементів на сторінці.
