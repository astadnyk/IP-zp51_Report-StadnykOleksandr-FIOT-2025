Контрольні питання та відповіді з CSS

  1. Для чого використовуються каскадні таблиці стилів?

    2. Каскадні таблиці стилів (CSS) використовуються для визначення зовнішнього вигляду HTML-документів. Вони дозволяють відокремити структуру контенту (HTML) від його презентації (CSS), що полегшує підтримку та зміну дизайну веб-сторінок.


  2. Які методи визначення стилів у HTML-документі ви знаєте?

    3. Існує три основні методи визначення стилів у HTML-документі:

    • Вбудовані стилі (inline styles): стилі, які застосовуються безпосередньо до HTML-елемента за допомогою атрибута style
    • Внутрішні стилі (internal styles): стилі, які визначаються всередині тегу <style> у розділі <head> HTML-документа
    • Зовнішні стилі (external styles): стилі, які зберігаються в окремому CSS-файлі і підключаються до HTML-документа за допомогою тегу <link>

  3. У чому різниця між каскадуванням і спадкуванням?

    4. Каскадування - це процес визначення пріоритету застосування стилів, коли кілька правил CSS можуть впливати на один і той же елемент. Правила з вищим пріоритетом (наприклад, більш специфічні селектори або стилі, визначені пізніше) переважають над іншими.

    Спадкування - це механізм, за допомогою якого деякі властивості CSS автоматично передаються від батьківського елемента до його дочірніх елементів. Наприклад, якщо встановити колір тексту для батьківського елемента, цей колір буде успадкований усіма його дочірніми елементами, якщо для них не задано інший колір.


  4. Як оформлюються селектори тегу і класу?

    5. Селектори тегу використовуються для застосування стилів до всіх елементів певного типу. Вони визначаються за допомогою імені HTML-тегу. Наприклад:

    h1 {
    font-size: 24px;
    color: blue;
}

    Цей стиль застосує розмір шрифту 24px і синій колір до всіх заголовків першого рівня<h1>

    Селектори класу використовуються для застосування стилів до елементів, які мають певний клас. Вони визначаються за допомогою крапки перед ім'ям класу. Наприклад:

    .highlight {
    background-color: yellow;
    font-weight: bold;
}

    Цей стиль застосує жовтий фон і жирний шрифт до всіх елементів з класом highlight


  5. Для чого використовується ідентифікатор?

    6. Ідентифікатор (ID) використовується для унікальної ідентифікації HTML-елемента на сторінці. Він дозволяє застосовувати стилі або взаємодіяти з конкретним елементом за допомогою JavaScript. Ідентифікатори визначаються за допомогою символу решітки (#) перед ім'ям ID. Наприклад:

    #header {
    background-color: lightblue;
    padding: 20px;
}

    Цей стиль застосує світло-блакитний фон і відступи до елемента з ID header


  6. Навести приклади оформлення текстових елементів.

    7. Ось кілька прикладів оформлення текстових елементів за допомогою CSS:

    p {
    font-size: 16px;
    line-height: 1.5;
    color: #333333;
    text-align: justify;
}
h2 {
    font-family: 'Arial', sans-serif;
    color: #0055cc;
    margin-bottom: 10px;
}
a {
    text-decoration: none;
    color: #ff6600;
}

    У цьому прикладі:

    • Абзаци (<p>) мають розмір шрифту 16px, міжрядковий інтервал 1.5, темно-сірий колір тексту і вирівнювання по ширині.
    • Заголовки другого рівня (<h2>) використовують шрифт Arial, синій колір і мають відступ знизу 10px.
    • Посилання (<a>) не мають підкреслення і мають помаранчевий колір.
