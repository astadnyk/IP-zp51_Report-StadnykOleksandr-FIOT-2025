Інші селектори

Окрім базових селекторів тегів, класів та ідентифікаторів, у CSS існує низка розширених селекторів, які дозволяють гнучко та точно керувати стилізацією елементів.

Селектори атрибутів

Селектори атрибутів дозволяють застосовувати стилі до елементів на основі наявності або значення певного атрибута.

input[type="text"] {
    border: 1px solid #ccc;
    padding: 5px;
}
a[href^="https://"] {
    color: green;
}
img[alt$="icon"] {
    width: 50px;
    height: 50px;
}

У цьому прикладі перший селектор стилізує всі текстові поля вводу, другий - всі посилання, що починаються з "https://", а третій - всі зображення, у яких атрибут alt закінчується на "icon".

Псевдокласи

Псевдокласи дозволяють стилізувати елементи в залежності від їх стану або позиції в документі.

a:hover {
    color: red;
}
li:nth-child(odd) {
    background-color: #f0f0f0;
}
button:disabled {
    background-color: #ccc;
}

У цьому прикладі перший селектор змінює колір посилання при наведенні курсору, другий - задає фон для непарних елементів списку, а третій - стилізує вимкнені кнопки.

Псевдоелементи

Псевдоелементи дозволяють стилізувати певні частини елементів, такі як перший рядок тексту або вміст перед або після елемента.

p::first-line {
    font-weight: bold;
}
div::before {
    content: "Note: ";
    font-style: italic;
}
span::after {
    content: " ✔";
    color: green;
}

У цьому прикладі перший селектор робить перший рядок абзацу жирним, другий додає текст "Note: " перед вмістом div, а третій додає зелений символ галочки після вмісту span.

Ці розширені селектори дозволяють більш точно налаштовувати вигляд веб-сторінок і покращують гнучкість у застосуванні стилів.
