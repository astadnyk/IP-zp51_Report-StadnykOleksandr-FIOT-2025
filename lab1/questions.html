Контрольні питання та відповіді

  1. Яка структура HTML-документу

    2. HTML документ починається з оголошення типу документа <!DOCTYPE html> за яким слідує кореневий елемент <html></html>
    Всередині нього є два основні розділи: <head></head> та <body></body>

  2. Дати визначення тега

    3. Тег - це спеціальний елемент HTML, який використовується для позначення початку та кінця різних частин контенту в документі. Наприклад, <p></p> <h1></h1> <img></img>

  3. Які теги використовуються при роботі з таблицями

    4. При роботі з таблицями використовуються такі теги: <table></table> <caption></caption> <thead></thead> <tbody></tbody> <tfoot></tfoot> <tr></tr> <th></th> <td></td>

  4. Які теги використовуються при роботі з зображеннями

    5. Для зображень використовується тег: <img></img> Атрибути: src alt width height

  5. Які теги використовуються при роботі зі списками

    6. При роботі зі списками використовуються такі теги: <ul></ul> <ol></ol> <li></li>

  6. Перерахувати засоби опису таблиць в HTML

    7. Засоби опису таблиць в HTML включають: <table></table> <caption></caption> <thead></thead> <tbody></tbody> <tfoot></tfoot> <tr></tr> <th></th> <td></td>
    Атрибути: colspan rowspan align valign
    CSS-властивості: border border-collapse padding text-align vertical-align
