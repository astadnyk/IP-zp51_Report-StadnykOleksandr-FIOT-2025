Опис курсу

Курс "Проектування WEB-застосувань" спрямований на навчання створенню сучасних веб-застосунків.

Під час виконання робіт, ви дізнаєтесь про основи веб-розробки, включаючи HTML, CSS, JavaScript та інші технології.

Інструкції для користувачів

  • Кожна лабораторна робота має окрему сторінку з детальним звітом.

  • Для перегляду звітів оберіть відповідну лабораторну роботу у верхньому меню.
